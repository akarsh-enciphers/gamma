Threads

A vulnerable web application developed by Enciphers

About

  • It is a Vulnerable Web Application build in Node.js for learning and practicing web application security. It is like a social media platform where people can connect with each other.
  • EJS is used as a frontend development templating engine with bootstrap and a mix of a simple HTML, CSS is used to build this vulnerable web application.
  • Express libraries which are used are
    1. Multer (file upload both with s3 bucket and disk storage).
    2. Passport.js (for cookie-based authentication and JWT authentication).
  • Basically this application contains various vulnerabilities as challenges for you to solve.

How to Install

  • Download: Click here
  • Go to side bar menu "Setup" tab to see how to configure the Threads application in your local system.

Features

This is like a social media application so it consist some of the similar features like:
  • A user can sign up and login into this application.
  • A user can create posts, he/she can also add images to the post he/she creates.
  • A user can like and can comment on other users posts.
  • A user can also chat with other users.
  • A user can tag as many users he want to tag in his post.

All Challenges

Low level severity

  • Self XSS
  • CSRF(Cross-site Request Forgery)
  • Hidden directories

Medium level severity

  • MongoDB Injection

High level severity

  • IDOR(Insecure Direct Object Reference)
  • Stored XSS
  • SSRF(Server-side Request Forgery)

Critical level severity

  • JWT Authentication
